Installation / Deployment

1. NodeJS (Server) : cd server

    2. npm install

    3. npm start

4. Angular 2 (Client) : cd client

    5. npm install

    6. npm start

7. Unit tests with database tests :

    8. cd server
    9. npm start
    10. npm test
